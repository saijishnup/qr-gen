# QR Generator

A lightweight, minimal QR code generator built for speed and simplicity.
No signup. No tracking. Just instant QR generation.

---

##  Features

- Instant QR generation
- Matte UI
- No backend
- Fully client-side

---

##  Project Structure

```
qr-generator/
│
├── index.html          # Main HTML file
├── theme.css        # Styling (UI + themes)
├── logic.js         # Core functionality
│
├── assets/
│   └── favicon.ico  # Favicon
│
├── libs/
│   └── qrcode.min.js  # QR generation library
│
├── README.md
└── LICENSE
```

## How to Run

### Option 1: Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/saijishnup/qr-gen.git
   ```

2. Open the project folder

3. Open the file:

   ```bash
   index.html
   ```

4. Open it in your browser

---

### Option 2: Use Live Version

Open directly in your browser:
 https://saijishnup.github.io/qr-gen/

## Tech Stack

* HTML5
* CSS3 (Glass UI + animations)
* Vanilla JavaScript
* QRCode.js (local library)

## Design Philosophy

This project focuses on:

* Minimal interface
* Fast load time
* Zero distractions
* Tool-first experience (not a platform)

---

## Usage

1. Enter text or URL
2. QR code is generated instantly
3. Download or copy as needed

---

## Notes

* Works fully offline
* No data is stored or sent anywhere
* No external dependencies (except:- local QR library)
